# open-in-browser README

Open HTML file in default browser


## USAGE

1, open command panel(`cmd + shift + p` on MacOS or `ctrl + shift + p` on Windows)
2, enter `open in browser`.

## SUPPORT

MacOS, Linux(firefox), Windows
