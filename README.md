# Open in Browser

This allows you to open the current file in your default browser or application.

## USAGE

> 1, open command panel ( `cmd + shift + p` on MacOS or `ctrl + shift + p` on Windows )
> 2, enter `open in browser`.
or 
with shortcut `Alt + B`



## SUPPORT

MacOS, Linux, Windows
